🔻 What Drains Health

  • Long conversation sessions (high token burn)
  • Frequent Telegram messages (each costs context)
  • Complex multi-step tasks (heavy processing)
  • Running out of API credits
  • Session restarts (lose context, rebuild costs)
  • Multiple concurrent tool calls

💚 How to Restore Health

  • Add API credits to your provider account
  • Use the dashboard instead of Telegram for status checks
  • Batch requests — fewer big asks > many small ones
  • Let Profit rest between heavy sessions
  • Use heartbeat system for automated checks
  • Keep conversations focused and concise
